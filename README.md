## Overview

NotePlan Professional Suite is a complete planning and note platform that merges calendar visualization, markdown writing, and project task management into a single daily workflow.

Time blocking displays scheduled tasks directly on a visual timeline alongside calendar events, making realistic daily planning possible by showing exactly how much time is actually available.

Project notes and folders organize longer-term work separately from daily notes while remaining cross-referenced, keeping ongoing projects connected to the daily entries where their tasks actually get done.

Markdown compatibility keeps every note in plain, portable text files synced through iCloud, ensuring the entire planning history remains readable and exportable independent of the application itself.

Tagging and mentions link tasks to specific projects or people across the entire note library, surfacing every reference to a given project regardless of which daily note it originally appeared in.

Themes and customizable editor styling adapt the writing environment's appearance, letting NotePlan Professional Suite match personal preference for color scheme and typography during extended daily planning sessions.

NotePlan Professional Suite is the platform for planners who think in both calendar time and written notes, and who want a single markdown-based tool that respects both.

## Performance & Integration

NotePlan is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while NotePlan processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
